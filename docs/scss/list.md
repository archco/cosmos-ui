# List
> source: [scss/parts/list](../../src/scss/parts/_list.scss)

### Usage
#### list
Simply none-list-style list.
```html
<ul class="list">
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ul>
```

#### list-group
```html
<ul class="list-group">
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ul>
```

#### list-horizontal
```html
<ul class="list-horizontal">
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ul>
```

#### breadcrumb
default separator is "/".
```html
<ul class="breadcrumb">
  <li><a href="#">scss</a></li>
  <li><a href="#">parts</a></li>
  <li>list</li>
</ul>
```
